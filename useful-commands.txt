# LIST OF GIT COMMANDS

git clone: copies the github project (repo) into a local folder in ur computer

git status: shows changes/updates n new files in project

git add FILENAME: includes the file changes to commit

git commit -m "DESCRIPTION": saves a snapshot of project with the changes that have been added

git log: displays a list of commits in project

git push: upload commits/changes to remote repository

git diff: tells u which changes have been made

git pull: pulls changes from online into local repo

node FILENAME: executes/runs js file
